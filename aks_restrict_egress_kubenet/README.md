## Install

``` powershell
terraform init
terraform apply --auto-approve
```

``` powershell
$resource_group=$(terraform output resource_group)
$aks_name=$(terraform output aks_name)
az aks get-credentials --resource-group $resource_group --name $aks_name
kubectl apply -f k8s/
```
